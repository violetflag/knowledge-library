# 대시보드

> 지식도서관 현황 대시보드

## 📊 통계

```dataview
TABLE topic AS 주제, count() AS 기록수
FROM "#경제 OR #건강 OR #마인드 OR #독서 OR #자녀교육 OR #한국역사 OR #세계사"
GROUP BY topic
SORT count() DESC
```

## 📅 최근 업데이트 (7일)

```dataview
TABLE date, topic, title
FROM "#경제 OR #건강 OR #마인드 OR #독서 OR #자녀교육 OR #한국역사 OR #세계사"
WHERE date >= this.week - 1
SORT date DESC
LIMIT 15
```

## ⏳ D-Days

```dataview
TABLE description AS 설명, target AS 대상, daysLeft AS "남은일수"
FROM "00_SYSTEM/D-Day.md"
WHERE daysLeft != null
SORT daysLeft ASC
```

## 🔥 최근 검색 기록

```dataview
TABLE query AS 검색어, time AS 검색시각
FROM "00_SYSTEM/log.md"
WHERE contains(text, "query")
SORT file.mtime DESC
LIMIT 10
```

## 📌 업데이트 필요 항목

```dataview
TABLE title, links FROM ; AND !outgoing
WHERE outgoing = 0 AND !file.name.match("D-Day|index|대시보드|log|settings")
SORT file.mtime ASC
LIMIT 10
```
