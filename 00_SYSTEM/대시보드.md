## 📈 주제별 통계 (Dataview)

```dataview
TABLE topic AS "주제", count(rows) AS "파일 수", sum(rows.wordCount) AS "단어 수"
FROM #경제 OR #건강 OR #마인드 OR #독서 OR #자녀교육 OR #한국역사 OR #세계사
GROUP BY topic
SORT topic
```

```dataviewjs
// Chart.js 주제별 파이차트
const topicCounts = {};
dv.pages('#경제 OR #건강 OR #마인드 OR #독서 OR #자녀교육 OR #한국역사 OR #세계사')
  .forEach(p => {
    const t = p.topic || "기타";
    topicCounts[t] = (topicCounts[t] || 0) + 1;
  });

const labels = Object.keys(topicCounts);
const data = Object.values(topicCounts);
const colors = {
  "경제": "#f39c12", "건강": "#27ae60", "마인드": "#9b59b6",
  "독서": "#e67e22", "자녀교육": "#1abc9c", "한국역사": "#c0392b", "세계사": "#34495e"
};

const config = {
  type: 'doughnut',
  data: {
    labels: labels,
    datasets: [{
      data: data,
      backgroundColor: labels.map(l => colors[l] || '#3498db'),
      borderWidth: 2,
      borderColor: '#fff'
    }]
  },
  options: {
    responsive: true,
    plugins: {
      legend: { position: 'bottom' },
      title: { display: true, text: '주제별 문서 분포' }
    }
  }
};

dv.span(`<canvas id="topicPie" style="max-height:300px;"></canvas>`);
dv.span(`<script>new Chart(document.getElementById('topicPie').getContext('2d'), ${JSON.stringify(config)});</script>`);
```

## ⏳ D-Day - 200세까지

```dataviewjs
const birthday = "1989-09-12";
const expectancy = 200;
const target = new Date(birthday);
target.setFullYear(target.getFullYear() + expectancy);
const now = new Date();
const diff = Math.floor((target - now) / (1000 * 60 * 60 * 24));
const yearsLeft = Math.floor(diff / 365);
const daysLeft = diff % 365;

dv.quote(`**${diff.toLocaleString()}일** 남음 (${yearsLeft}년 ${daysLeft}일)`);
```

## 📈 월별 기록 추이 (Charts)

```dataviewjs
// 월별 문서 수 집계
const monthly = {};
dv.pages('#경제 OR #건강 OR #마인드 OR #독서 OR #자녀교육 OR #한국역사 OR #세계사')
  .forEach(p => {
    const month = p.file.mday.slice(0, 7); // YYYY-MM
    monthly[month] = (monthly[month] || 0) + 1;
  });

const sortedMonths = Object.keys(monthly).sort();
const counts = sortedMonths.map(m => monthly[m]);

const lineConfig = {
  type: 'line',
  data: {
    labels: sortedMonths,
    datasets: [{
      label: '문서 수',
      data: counts,
      borderColor: '#3498db',
      backgroundColor: 'rgba(52, 152, 219, 0.1)',
      fill: true,
      tension: 0.3
    }]
  },
  options: {
    responsive: true,
    plugins: {
      title: { display: true, text: '월별 기록 추이' }
    },
    scales: {
      y: { beginAtZero: true, ticks: { stepSize: 1 } }
    }
  }
};

dv.span(`<canvas id="monthlyChart"></canvas>`);
dv.span(`<script>new Chart(document.getElementById('monthlyChart').getContext('2d'), ${JSON.stringify(lineConfig)});</script>`);
```
