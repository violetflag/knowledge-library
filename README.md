# Obsidian Bolt: 옵시디언
## Setup Guide

1.  **Dashboard:** Open `Dashboard/Home.md` to see the executive overview.
2.  **Plugins:** Recommend using the following plugins for the best experience:
    - **Dataview:** To automate the KPI and project tracking.
    - **Calendar:** To view the calendar view directly in Obsidian.
    - **Kanban:** For managing project tasks visually.
3.  **Links:** Use `[[ ]]` to create links between your notes and the dashboard.
